# DisplayXR Unity Plugin

Unity engine plugin for [DisplayXR](https://github.com/DisplayXR/displayxr-runtime), distributed as a UPM (Unity Package Manager) package.

## Status

This repo will replace `dfattal/unity-3d-display`. Migration pending.

## Installation

Install via UPM git URL (once migrated):
```
https://github.com/DisplayXR/displayxr-unity.git
```
